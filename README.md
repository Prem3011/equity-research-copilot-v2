# AI Equity Research Copilot v2

Professional-grade AI equity research briefs for US and Indian stocks. Real financial data + AI-powered analysis in seconds.

**[Live Demo →](https://equity-research-copilot-v2.vercel.app)**

---

## What It Does

Enter any stock ticker (AAPL, RELIANCE, TCS, NVDA...) and get a complete equity research report including:

- **5-year financial table** — Revenue, EBITDA, CFO, FCF, leverage ratios, EPS (real data from company filings)
- **FCF waterfall chart** — Moody's-style stacked bar + line overlay
- **Revenue breakdown** — By business segment and geography (pie charts)
- **Debt maturity profile** — Stacked bar chart by year
- **Liquidity analysis** — Sources vs Uses of cash
- **Risk factors** — Key business and market risks
- **Bull vs Bear case** — Balanced investment view
- **Investment thesis** — AI-generated conclusion with outlook rating
- **Latest news** — Recent headlines with clickable source links
- **Peer comparison** — Suggested comparable companies

## Architecture

```
User enters ticker
        ↓
  Next.js API Route (/api/analyze)
        ↓
  ┌─────┴─────┐
  ↓           ↓
FMP API    Gemini API
(financials) (AI analysis)
  ↓           ↓
  └─────┬─────┘
        ↓
  Combined JSON response
        ↓
  React components render
  charts + tables + cards
```

**FMP API** provides 5 years of audited income statements, balance sheets, and cash flow statements. Supports both US (NYSE/NASDAQ) and Indian (NSE) stocks.

**Gemini API** with Google Search grounding generates qualitative analysis — company overview, risk factors, investment thesis, news headlines, and revenue/geographic breakdowns.

All derived metrics (EBITDA, FCF, leverage ratios) are computed from raw financial statement data, not AI estimates.

## Getting Started

### Prerequisites
- Node.js 18+
- Free API keys from:
  - [Financial Modeling Prep](https://site.financialmodelingprep.com/register)
  - [Google AI Studio](https://aistudio.google.com)

### Local Development

```bash
git clone https://github.com/YOUR_USERNAME/equity-research-copilot-v2.git
cd equity-research-copilot-v2
npm install
```

Create `.env.local`:
```
FMP_API_KEY=your_fmp_key
GEMINI_API_KEY=your_gemini_key
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Deploy to Vercel

1. Push to GitHub
2. Import to [Vercel](https://vercel.com)
3. Add environment variables: `FMP_API_KEY` + `GEMINI_API_KEY`
4. Deploy
